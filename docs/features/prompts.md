# Prompts

The prompt input area is the primary way you communicate with your agents. From typing a quick instruction to dictating a multi-step task, every message you send — text, voice, or image — flows through the prompt area at the bottom of the SquadDash window.

## In This Chapter

- **[Entering Prompts](entering-prompts.md)** — How the prompt text box works, sending and queuing prompts, prompt history, slash commands, and `@agent` routing.
- **[Attaching Clipboard Text](attach-clipboard-text.md)** — Use **Ctrl+Shift+V** to attach large text (logs, error output, code blocks) as a follow-up attachment instead of pasting it inline.
- **[Paste Image](paste-image.md)** — Paste a screenshot or image directly from the clipboard into a prompt.
- **[Voice Input](voice-input.md)** — Dictate prompts with push-to-talk using your microphone.
- **[Prompt Queue](prompt-queue.md)** — Queue up prompts while an agent is running so they execute in order.
